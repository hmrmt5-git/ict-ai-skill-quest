# AGENTS.md

- UIは日本語を基本とし、教職員と高校生の双方に分かりやすくする。
- 秘密情報、OAuthクライアントシークレット、サービスアカウント鍵、正式ロゴ未公開素材をコミットしない。
- `.env.example` 以外の環境変数ファイルをコミットしない。
- `wasegaku.ac.jp` 制限はフロントエンドだけでなくサーバー側でも検証する。
- 問題、採点、合否、挑戦回数、修了証番号、管理者権限をクライアント側だけで処理しない。
- コード変更後は `npm run lint`、`npm run build`、`npm test` を実行する。
- 利用者向け動作を変更した場合は関連文書も更新する。
- 挙動を変更した場合は関連ドキュメントも更新する。
- サンプル問題には「試作用仮問題・本番利用不可」を残し、本番問題として扱わない。
- 既存のセキュリティ要件を弱めず、アクセシビリティを後退させない。
- ライト・ダーク双方、サウンドON・OFF双方、動きを減らす設定を確認する。
- GitHub Pagesのサブパスでの動作を確認する。
- 新しい依存ライブラリは必要性を検討してから追加する。
- 過剰な抽象化を避け、保守しやすいコードを書く。
- 作業完了前およびプルリクエスト前に差分を自己レビューする。
